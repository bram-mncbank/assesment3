# Assesment3
## How to install
~~~
git clone https://github.com/bram-mncbank/assesment3
~~~

## How to Run
### Run WEB server
Enter directory and run the main.go:
~~~
cd assesment3
go run main.go
~~~
### How to access
1. API weather can access at:
~~~
http://localhost:8080/weather
~~~
2. For data presentation, there is a page can visit:
~~~
http://localhost:8080
~~~
